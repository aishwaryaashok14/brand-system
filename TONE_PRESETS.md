# Tone Presets Reference

Ten curated brand directions. Each preset is a starting point — composite or fork freely. **No illustrations, no stock photography. Imagery is described as concrete reference scenes.**

Use the **Cultural Reference** line to ground each direction in something real. If the user knows the reference, the rest follows.

---

## 1. Quiet Editorial

**Cultural reference:** Print magazines (The Gentlewoman, Apartamento), academic press, slow journalism

**Vibe:** Considered, literary, precise. Confident enough to whisper.

**Palette:**
```css
--bone:   #f4f1ea;  /* page */
--ink:    #14110f;  /* primary text */
--ash:    #6e6862;  /* secondary text */
--rust:   #b04a2c;  /* sole accent */
--moss:   #4a5743;  /* cool counter */
--paper:  #e9e3d6;  /* surface */
```

**Type:**
- Display: `Fraunces` 700 (variable, 14pt optical) or `PP Editorial Old`
- Body: `Söhne` 400/500 or `Inter Tight` 400/500
- Mono (rare): `JetBrains Mono` 400

**Voice:** Declarative, specific, never breathless. Sentences end early.

**Imagery anchors:**
- Hands holding a single object (book, ceramic, fruit)
- Window light, late morning, soft shadow
- Materials: linen, oak, unglazed clay, hand-set type, deckle paper

**Reject:** stock office scenes, lifestyle composites, gradient meshes

## 2. Field Manual

**Cultural reference:** US Army TM-31 manuals, Braun product spec sheets, vintage tool catalogs

**Vibe:** Honest, technical, blue-collar precision. Built to be read on a job site.

**Palette:**
```css
--ink:      #0d0c0a;
--paper:    #f1ede4;
--hazard:   #ffb800;  /* signal */
--steel:    #6a6e74;
--rust:     #8a3a1a;
--graphite: #3a3a3c;
```

**Type:**
- Display: `Archivo Black` 900 or `Söhne Breit` (condensed/extended)
- Body: `Söhne Mono` 400 or `JetBrains Mono` 400/700
- Numerals: tabular, always

**Voice:** Imperative. Numbered. "Tighten clamp to 8 nm. Do not exceed."

**Imagery anchors:**
- Tool laid flat on grid paper
- Diagrams over photographs
- Materials: galvanized steel, Manila card, hi-vis tape, exposed fasteners

**Reject:** soft drop-shadows, gradients, smile photography

## Voice Words That Almost Always Work

Specific nouns ("rye flour," "8mm clamp," "blue hour") > generic adjectives ("amazing," "elevated"). Active verbs > nominalizations ("ship" > "deliver shipments"). Concrete time ("by 6am") > vague time ("quickly").

## Voice Words to Reject Across All Presets

empower · elevate · seamless · delight · unleash · revolutionize · transform · curate (when used as filler) · bespoke · artisanal · journey · solutions · cutting-edge · world-class · best-in-class

---

## Composing a Direction Outside These Presets

When none fit, build from these axes:
1. **Cultural reference** — name a real thing (magazine, store, album, building, community)
2. **One-word vibe** — single adjective that the palette + type + voice all serve
3. **Palette skeleton** — 1 ground (bg), 1 ink (text), 1 signal (accent that does work), 2 quiet supporting tones, 1 deep accent (shadow/oxblood/moss)
4. **Type pair** — 1 display with personality, 1 body that disappears
5. **Imagery anchor sentence** — "subject in [environment] under [light condition]"
6. **What it rejects** — name 3 specific things this brand will not do

If you can't fill all six lines, the direction isn't crisp enough yet.
